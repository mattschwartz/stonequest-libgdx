# StoneQuest #

## ABOUT ##
StoneQuest is a 2D, top-down aRPG written in Java on top of the LWJGL system.

## LIBRARIES USED ##
StoneQuest takes advantage of several free, third-party libraries:
* LWJGL
* libgdx
* box2d and box2dlights
